# Καθολική Αναζήτηση - Οδηγός

Οδηγός λειτουργίας και τεχνικής υλοποίησης της καθολικής αναζήτησης του CforC website.

---

## Επισκόπηση

Η καθολική αναζήτηση είναι ένα modal παράθυρο που επιτρέπει στους χρήστες να αναζητούν περιεχόμενο σε ολόκληρο τον ιστότοπο. Ανοίγει με:

- **Εικονίδιο αναζήτησης** στο navigation bar (αριστερά του κουμπιού dark/light mode)
- **Συντόμευση πληκτρολογίου** `Cmd+K` (Mac) ή `Ctrl+K` (Windows/Linux)
- **Κουμπί ΑΝΑΖΗΤΗΣΗ** στο mobile menu

---

## Κατηγορίες Αναζήτησης

Η αναζήτηση καλύπτει 4 κατηγορίες περιεχομένου:

| Κατηγορία | Ετικέτα | Πηγή Δεδομένων | Πρόσβαση |
|-----------|---------|----------------|----------|
| Μέλη | Μέλη | Strapi - Members | Όλοι |
| Νέα | Νέα | Strapi - Activities | Όλοι |
| Ανοιχτές Προσκλήσεις | Ανοιχτές Προσκλήσεις | Strapi - Open Calls | Μόνο μέλη |
| Σελίδες | Σελίδες | Στατικό (hardcoded) | Όλοι |

### Πεδία που αναζητούνται ανά κατηγορία

| Κατηγορία | Πεδία Strapi |
|-----------|-------------|
| **Μέλη** | `Name`, `FieldsOfWork`, `City` |
| **Νέα** | `Title`, `EngTitle`, `Category` |
| **Ανοιχτές Προσκλήσεις** | `Title`, `EngTitle`, `Category` |
| **Σελίδες** | Τίτλος σελίδας + λέξεις-κλειδιά (keywords) |

> **Σημείωση:** Τα πεδία `Description` και `EngDescription` είναι τύπου rich text (Strapi blocks) και δεν υποστηρίζουν αναζήτηση μέσω Strapi API filters.

## Φίλτρα (Chips)

Τα φίλτρα εμφανίζονται ως toggle chips κάτω από το πεδίο αναζήτησης:

### Εμφάνιση chip

| Κατάσταση | Light Theme | Dark Theme |
|-----------|-------------|------------|
| **Ενεργό** | Μαύρο φόντο, λευκό κείμενο | Λευκό φόντο, μαύρο κείμενο |
| **Ανενεργό** | Λευκό φόντο, μαύρο κείμενο | Σκούρο φόντο, λευκό κείμενο |
| **Κλειδωμένο** (Ανοιχτές Προσκλήσεις - μη μέλος) | Γκρι κείμενο, εικονίδιο κλειδαριάς | Γκρι κείμενο, εικονίδιο κλειδαριάς |

### Κουμπιά ελέγχου
- **Επιλογή όλων** — Ενεργοποιεί όλες τις διαθέσιμες κατηγορίες
- **Αποεπιλογή** — Απενεργοποιεί όλες εκτός από «Σελίδες» (ελάχιστη 1 κατηγορία)

### Υπόμνημα
Κάτω από τα chips εμφανίζεται οπτικό υπόμνημα (legend) που δείχνει ποιο χρώμα σημαίνει ενεργό/ανενεργό, προσαρμοσμένο στο τρέχον theme.

## Ανοιχτές Προσκλήσεις — Πρόσβαση Μελών

Η αναζήτηση στις Ανοιχτές Προσκλήσεις είναι διαθέσιμη **μόνο για εγγεγραμμένα μέλη**.

### Μη συνδεδεμένος χρήστης
- Το chip εμφανίζεται με εικονίδιο κλειδαριάς και γκρι χρώματα
- **Hover:** Εμφανίζεται tooltip «Διαθέσιμο μόνο για εγγεγραμμένα μέλη» + το κείμενο αλλάζει σε **«Κλικ για σύνδεση»**
- **Κλικ:** Ανακατεύθυνση στη σελίδα σύνδεσης (`/login?returnTo=ΤΡΕΧΟΥΣΑ_ΣΕΛΙΔΑ?search=1`)
- **Μετά τη σύνδεση:** Ο χρήστης επιστρέφει στη σελίδα που ήταν με το modal αναζήτησης ανοιχτό

### Συνδεδεμένος χρήστης
- Το chip λειτουργεί κανονικά (toggle on/off)
- Τα αποτελέσματα περιλαμβάνουν Ανοιχτές Προσκλήσεις

## Αποτελέσματα

### Εμφάνιση
- Ομαδοποιημένα κατά κατηγορία με header (πχ. «ΜΕΛΗ (3)»)
- Κάθε αποτέλεσμα εμφανίζει:
  - **Μέλη:** Avatar, όνομα, πεδίο εργασίας, πόλη
  - **Νέα:** Εικονίδιο, τίτλος, κατηγορία, ημερομηνία
  - **Ανοιχτές Προσκλήσεις:** Εικονίδιο, τίτλος, κατηγορία, deadline
  - **Σελίδες:** Εικονίδιο, τίτλος σελίδας

### Πλοήγηση
- **Βέλη ↑↓:** Πλοήγηση μεταξύ αποτελεσμάτων
- **Enter:** Άνοιγμα επιλεγμένου αποτελέσματος
- **Κλικ:** Άνοιγμα αποτελέσματος
- **ESC:** Κλείσιμο modal

### Κενά αποτελέσματα
- Μήνυμα: «Δεν βρέθηκαν αποτελέσματα για "ΟΡΟΣ"»
- Υπότιτλος: «Δοκίμασε διαφορετικούς όρους αναζήτησης»

## Τεχνική Υλοποίηση

### Αρχεία

| Αρχείο | Ρόλος |
|--------|-------|
| `components/GlobalSearch.tsx` | Modal component (UI, chips, αποτελέσματα, keyboard navigation) |
| `app/api/search/route.ts` | API endpoint - αναζήτηση σε Strapi + στατικές σελίδες |
| `components/Navigation.tsx` | Ενσωμάτωση search icon, Cmd+K shortcut, auto-open από URL param |

### API Route: `/api/search`

**Μέθοδος:** GET

**Παράμετροι:**

| Παράμετρος | Τύπος | Περιγραφή |
|------------|-------|-----------|
| `q` | string | Όρος αναζήτησης (ελάχιστο 2 χαρακτήρες) |
| `types` | string | Κατηγορίες (comma-separated): `members,activities,open-calls,pages` |
| `includeOpenCalls` | string | `true` / `false` — ελέγχεται από auth status |

**Απάντηση:**
```json
{
  "members": [...],
  "activities": [...],
  "openCalls": [...],
  "pages": [...]
}
```

### Debounce
Η αναζήτηση εκτελείται με debounce **300ms** — δηλαδή περιμένει 300ms μετά τον τελευταίο χαρακτήρα πριν στείλει αίτημα.

### Στατικές σελίδες
Οι στατικές σελίδες αναζητούνται τοπικά (χωρίς API call) με αντιστοίχιση τίτλου και λέξεων-κλειδιών:

| Σελίδα | URL |
|--------|-----|
| Σχετικά με εμάς | `/about` |
| Ομάδα Συντονισμού | `/coordination-team` |
| Διαφάνεια | `/transparency` |
| Συμμετοχή | `/participation` |
| Επικοινωνία | `/contact` |
| Πολιτική Απορρήτου | `/privacy` |
| Όροι Χρήσης | `/terms` |
| Cookies | `/cookies` |
| Προσβασιμότητα | `/accessibility` |
