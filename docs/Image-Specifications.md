# CforC Website - Image Specifications

Comprehensive guide for all image requirements across the CforC website and Strapi CMS.

---

## Νέα / Δραστηριότητες (News / Activities)

### Thumbnail (Κάρτα Προεπισκόπησης)
- **Αναλογίες:** 16:9 (π.χ. 1920x1080 ή 1280x720)
- **Μέγεθος:** μέγιστο 150-250KB
- **Χρήση:** Κάρτες νέων στην αρχική σελίδα, στη σελίδα νέων (grid & list view) και στα σχετικά νέα κάτω από κάθε άρθρο.
- **Σημείωση:** Η σημαντική πληροφορία να βρίσκεται στο κέντρο της εικόνας. Η εικόνα κόβεται με `object-cover`.
- **Strapi field:** `Visuals` (πρώτη εικόνα χρησιμοποιείται ως thumbnail)

### Εικόνες Άρθρου (Article Detail)
- **Αναλογίες:** 16:9 (π.χ. 1920x1080)
- **Μέγεθος:** μέγιστο 150-250KB/εικόνα
- **Χρήση:** Κύρια εικόνα και carousel στη σελίδα λεπτομερειών του νέου.
- **Σημείωση:** Υποστηρίζονται πολλαπλές εικόνες. Thumbnails στο carousel εμφανίζονται σε 96x96px.

### Inline Εικόνες (Μέσα στο κείμενο)
- **Αναλογίες:** Ελεύθερο
- **Μέγεθος:** μέγιστο 150-250KB/εικόνα
- **Χρήση:** Εικόνες μέσα στο rich text περιεχόμενο του άρθρου.
- **Σημείωση:** Οι εικόνες μεγαλώνουν αυτόματα και αναλογικά μέχρι να καλύψουν τον προκαθορισμένο χώρο τους.

## Ανοιχτές Προσκλήσεις (Open Calls)

### Thumbnail (Κάρτα Προεπισκόπησης)
- **Αναλογίες:** 16:9 (π.χ. 1920x1080 ή 1280x720)
- **Μέγεθος:** μέγιστο 150-250KB
- **Χρήση:** Κάρτες ανοιχτών προσκλήσεων στην αρχική σελίδα και στη σελίδα ανοιχτών προσκλήσεων (grid & list view).
- **Σημείωση:** Η σημαντική πληροφορία να βρίσκεται στο κέντρο. Αποφύγετε λογότυπα ή banners ως cover image - χρησιμοποιήστε αντιπροσωπευτική φωτογραφία.
- **Strapi field:** `Image`

## Μέλη (Members)

### Φωτογραφία Προφίλ (Profile Image)
- **Αναλογίες:** 5:6 (π.χ. 1080x1296 ή 900x1080)
- **Μέγεθος:** μέγιστο 150-250KB
- **Χρήση:** Κάρτες μελών στη σελίδα μελών (grid view), στη σελίδα επεξεργασίας προφίλ και στο profile preview modal.
- **Σημείωση:** Κατά προτίμηση το πρόσωπο να βρίσκεται στο κέντρο της εικόνας. Portrait φωτογραφία.
- **Strapi field:** `Image`

### Φωτογραφία Προφίλ (Detail Page)
- **Αναλογίες:** 3:4 (π.χ. 900x1200 ή 810x1080)
- **Μέγεθος:** μέγιστο 150-250KB
- **Χρήση:** Κύρια φωτογραφία στη σελίδα λεπτομερειών μέλους.
- **Σημείωση:** Η ίδια εικόνα χρησιμοποιείται και στα δύο (card & detail) - η εικόνα κόβεται αυτόματα. Φροντίστε το πρόσωπο να είναι κεντραρισμένο.

### Avatar (List View & Ομάδες Εργασίας)
- **Αναλογίες:** 1:1 (π.χ. 200x200)
- **Μέγεθος:** μέγιστο 50KB
- **Χρήση:** Μικρό avatar στο list view μελών (64x64px), στις ομάδες εργασίας (48x48px ή 32x32px) και στα projects.
- **Σημείωση:** Αυτόματα κόβεται σε κύκλο. Η ίδια εικόνα προφίλ χρησιμοποιείται - δεν χρειάζεται ξεχωριστό upload.

## Projects (Έργα CforC)

### Cover Image (Εξώφυλλο)
- **Αναλογίες:** 16:9 (π.χ. 1920x1080 ή 1280x720)
- **Μέγεθος:** μέγιστο 150-250KB
- **Χρήση:** Κύρια εικόνα/carousel στη σελίδα λεπτομερειών project.
- **Σημείωση:** Η σημαντική πληροφορία να βρίσκεται στο κέντρο. Thumbnails στο carousel εμφανίζονται σε 80x80px.
- **Strapi field:** `cover_image`

### Project Images (Πρόσθετες Εικόνες)
- **Αναλογίες:** 16:9 (π.χ. 1920x1080)
- **Μέγεθος:** μέγιστο 150-250KB/εικόνα
- **Χρήση:** Carousel εικόνων στο project detail.
- **Strapi field:** `project_images`

### Supporters Banner
- **Αναλογίες:** Ελεύθερο (συνήθως πολύ φαρδύ, π.χ. 1200x200)
- **Μέγεθος:** μέγιστο 150KB
- **Χρήση:** Banner υποστηρικτών στο κάτω μέρος του project. Υπάρχουν ξεχωριστά για light και dark mode.
- **Strapi fields:** `supporters_banner_light`, `supporters_banner_dark`

- **Μέγεθος:** μέγιστο 150-250KB/εικόνα

- **Σημείωση:** Υποστηρίζονται πολλαπλές εικόνες ανά project.

## Coordination Team (Ομάδα Συντονισμού)

### Team Banner
- **Αναλογίες:** 3:1 (π.χ. 1500x500 ή 1200x400)
- **Μέγεθος:** μέγιστο 150-250KB
- **Χρήση:** Banner εικόνα στο πάνω μέρος κάθε ομάδας συντονισμού.
- **Σημείωση:** Wide landscape format. Η εικόνα κόβεται με `object-cover`.

## Στατικές Σελίδες (Homepage & About)

### Hero Image (About Page)
- **Αναλογίες:** 16:9 (π.χ. 1920x1080)
- **Μέγεθος:** μέγιστο 250-400KB (μεγαλύτερο λόγω full-screen)
- **Χρήση:** Full-height parallax background στη σελίδα "Σχετικά με εμάς". Καταλαμβάνει 80% του viewport.
- **Σημείωση:** Χρησιμοποιήστε εικόνα υψηλής ποιότητας. Η σημαντική πληροφορία να βρίσκεται στο κέντρο.

## Γενικές Οδηγίες

### Format
- **Προτιμώμενα formats:** WebP, JPEG, PNG
- **WebP** προτιμάται για καλύτερη συμπίεση και ταχύτερο loading

### Βελτιστοποίηση
- Πάντα βελτιστοποιήστε τις εικόνες πριν τις ανεβάσετε στο Strapi
- Η εικόνα κόβεται αυτόματα με `object-cover` - φροντίστε η σημαντική πληροφορία να είναι κεντραρισμένη
- Για portrait εικόνες (μέλη), το πρόσωπο πρέπει να βρίσκεται στο κέντρο

### Alt Text (Εναλλακτικό Κείμενο)
- **Υποχρεωτικό** για κάθε εικόνα (πεδίο `ImageAltText` ή `ProfileImageAltText` στο Strapi)
- Περιγράψτε συνοπτικά τι απεικονίζει η εικόνα
- Χρησιμοποιείται για προσβασιμότητα (screen readers) και SEO

2. **Optimization:** https://imagecompressor.com/
3. **Edit & Optimization:** https://tinywow.com/
