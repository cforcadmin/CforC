# Οδηγός Έργων και Καταχωρήσεων

## Πώς λειτουργεί

Το σύστημα έχει δύο επίπεδα:

- **Έργο (Project)** = ένας "φάκελος" (π.χ. ΣΗΜΑ, Humanitarium)
- **Καταχώρηση Έργου (Project Entry)** = μια σελίδα/άρθρο μέσα σε αυτόν τον φάκελο

Στο website εμφανίζονται:
- `/projects` — Λίστα με όλα τα έργα
- `/projects/sima` — Η σελίδα του έργου ΣΗΜΑ με τις δημόσιες καταχωρήσεις του
- `/projects/sima/entries/katagrafes-2026` — Μια μεμονωμένη καταχώρηση

---

## Πώς να προσθέσετε ένα νέο Έργο

1. Συνδεθείτε στο Strapi admin panel
2. Μεταβείτε στο **Content Manager** → **Project**
3. Πατήστε **Create new entry**
4. Συμπληρώστε τα πεδία:
   - **title**: Τίτλος του έργου (υποχρεωτικό)
   - **slug**: Δημιουργείται αυτόματα από τον τίτλο — χρησιμοποιείται στο URL
   - **short_description**: Σύντομη περιγραφή (εμφανίζεται στις κάρτες και στο dropdown πλοήγησης)
   - **full_description**: Πλήρης περιγραφή (rich text, εμφανίζεται στη σελίδα του έργου)
   - **cover_image**: Κύρια εικόνα (εμφανίζεται σε κάρτες και hero)
   - **project_images**: Επιπλέον φωτογραφίες (γκαλερί στη σελίδα του έργου)
   - **project_link**: Εξωτερικός σύνδεσμος ιστοσελίδας (προαιρετικό — κρύβεται αν είναι κενό)
   - **project_status**: `active`, `in_progress`, ή `completed`
   - **start_date** / **end_date**: Ημερομηνίες έργου
   - **category**: Tags χωρισμένα με κόμμα, π.χ. `AI, culture, education`
   - **sort_order**: Αριθμός για τη σειρά εμφάνισης (μικρότερος = πρώτα)
   - **featured**: Αν είναι `true`, εμφανίζεται στο dropdown πλοήγησης
   - **partners**: Επαναλαμβανόμενο component (όνομα, λογότυπο, URL)
   - **external_links**: Επαναλαμβανόμενο component (ετικέτα, URL)
5. Πατήστε **Save** και μετά **Publish**

---

## Πώς να προσθέσετε μια Καταχώρηση Έργου

1. Μεταβείτε στο **Content Manager** → **Project Entry**
2. Πατήστε **Create new entry**
3. Συμπληρώστε τα πεδία:
   - **title**: Τίτλος (υποχρεωτικό)
   - **slug**: Δημιουργείται αυτόματα από τον τίτλο
   - **description**: Περιγραφή (rich text)
   - **cover_image**: Κύρια εικόνα
   - **images**: Επιπλέον φωτογραφίες (γκαλερί)
   - **category**: Κατηγορία (π.χ. Έρευνα, Ανακοίνωση)
   - **tags**: Tags χωρισμένα με κόμμα, π.χ. `σημα, cforc, bodosaki`
   - **entry_link**: Εξωτερικός σύνδεσμος (προαιρετικό — κρύβεται αν είναι κενό)
   - **visibility**: `public` ή `private` (βλ. παρακάτω)
   - **publication_date**: Ημερομηνία δημοσίευσης
   - **expiration_date**: Ημερομηνία λήξης (προαιρετικό)
   - **project**: Επιλέξτε σε ποιο Έργο ανήκει
4. Πατήστε **Save** και **Publish**

---

## Visibility: Public vs Private

- **public**: Η καταχώρηση εμφανίζεται στη σελίδα του έργου και στα αποτελέσματα αναζήτησης
- **private**: Η καταχώρηση ΔΕΝ εμφανίζεται στη σελίδα του έργου. Είναι προσβάσιμη μόνο μέσω direct link: `/projects/[slug-εργου]/entries/[slug-καταχωρησης]`. Οι μηχανές αναζήτησης δεν την ευρετηριάζουν (noindex).

### Πώς να κάνετε μια private καταχώρηση public:
1. Μεταβείτε στην καταχώρηση στο Strapi
2. Αλλάξτε το πεδίο **visibility** από `private` σε `public`
3. Πατήστε **Save**

---

## Σύστημα Λήξης (Expiration)

Αν ορίσετε **expiration_date** και αυτή περάσει:
- Η σελίδα εξακολουθεί να είναι προσβάσιμη
- Αντί για το περιεχόμενο, εμφανίζεται μήνυμα: "Αυτή η καταχώρηση δεν είναι πλέον διαθέσιμη"

---

## Πώς να αλλάξετε τη σειρά των Έργων

1. Ανοίξτε κάθε Έργο στο Strapi
2. Τροποποιήστε το πεδίο **sort_order**
3. Μικρότερος αριθμός = εμφανίζεται πρώτα (0, 1, 2, ...)
4. Η σειρά ισχύει τόσο στο dropdown πλοήγησης όσο και στη σελίδα λίστας

---

Αυτή τη στιγμή, το "ΕΡΓΑ" εμφανίζεται μόνο όταν ο χρήστης βρίσκεται στη διαδρομή `/projects` (λειτουργία δοκιμών).

Για να γίνει μόνιμα ορατό:
1. Ανοίξτε το αρχείο `components/Navigation.tsx`
2. Βρείτε τη γραμμή: `// TODO: Remove testing condition — make permanently visible`
3. Αλλάξτε `const showProjects = pathname?.startsWith('/projects')` σε `const showProjects = true`
4. Αφαιρέστε τα σχόλια TODO
