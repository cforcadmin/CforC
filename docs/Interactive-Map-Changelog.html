1. Νέα Σελίδα Χάρτη (/map) NEW

Επισκόπηση

Πλήρης διαδραστική σελίδα χάρτη Ελλάδας με SVG choropleth, zoom, avatar clusters, ένθετο χάρτη Ευρώπης, dark mode. Βασίζεται σε react-simple-maps + GeoJSON δεδομένα Eurostat NUTS2.

Τεχνολογίες

ΠακέτοΧρήση

d3-geoProjection utilities
GeoJSON (Eurostat NUTS2)Γεωμετρία 13 περιφερειών Ελλάδας

Persistent Search State (sessionStorage)

Η αναζήτηση/φίλτρα αποθηκεύονται στο sessionStorage και επαναφέρονται κατά την επιστροφή.

ΣελίδαΚλειδίΑποθηκεύειΑρχείο
Μέλη (/members) cforc-members-search searchQuery, selectedFields, selectedCities, selectedProvinces, sortMode, filterLogic, viewMode app/members/page.tsx
Νέα (/news) cforc-news-search searchQuery, activeTab, selectedYear app/news/page.tsx
Ανοιχτές Προσκλήσεις (/open-calls) cforc-opencalls-search searchQuery, activeTab, selectedYear components/OpenCallsContent.tsx
Προφίλ (/profile) cforc-profile-section Ενεργή ενότητα dashboard app/profile/page.tsx
Προτεραιότητα: URL params > sessionStorage > defaults. Το κουμπί "Καθαρισμός" διαγράφει και το sessionStorage.

5. Τεχνικές Σημειώσεις FIXES

Hydration Fixes

ΠρόβλημαΛύση
react-simple-maps markers παράγουν floating-point transform attributes διαφορετικά σε server/client Client-only rendering (mounted state via useEffect) στο EuropeInset
Google Translate widget εισάγει role/tabindex πριν το hydration suppressHydrationWarning στο LanguageSwitcher container
Nested <button> σε <button> (CategoryFilter, YearFilter) Αντικατάσταση εσωτερικού <button> με <span role="button">

Peer Dependency Fix

Πρόβλημα: react-simple-maps@3.0.0 απαιτεί react@^16.8.0 || 17.x || 18.x αλλά το project χρησιμοποιεί React 19
Λύση: .npmrc με legacy-peer-deps=true

Year Persistence Bug Fix

Πρόβλημα: Το φίλτρο έτους δεν αποθηκευόταν σωστά (σε Νέα & Ανοιχτές Προσκλήσεις)
Αιτία: React effects τρέχουν κατά σειρά δήλωσης — restore effect + reset effect δημιουργούσαν race condition
Λύση: Αντικατάσταση effect-based reset με handleTabChange callback wrapper
Pattern: Μην χρησιμοποιείτε effects για state resets που εξαρτώνται από άλλα state updates στο ίδιο render cycle. Χρησιμοποιήστε callback wrappers αντί αυτού.
