# Οδηγός Ενσωμάτωσης Εικόνων για Συντάκτες Περιεχομένου

## Επισκόπηση

Ο ενσωματωμένος επεξεργαστής Blocks του Strapi δεν υποστηρίζει την εισαγωγή εικόνων μέσα σε πεδία rich text. Για να το αντιμετωπίσουμε αυτό, χρησιμοποιούμε μια απλή σύμβαση κειμένου: πληκτρολογείτε μια ειδική ετικέτα σε μια παράγραφο και η ιστοσελίδα την εμφανίζει αυτόματα ως εικόνα.

Αυτό λειτουργεί σε **οποιοδήποτε πεδίο rich text** σε ολόκληρο τον ιστότοπο: περιγραφές δραστηριοτήτων, περιγραφές έργων, βιογραφικά μελών, καταχωρήσεις έργων κ.λπ.

---

## Βασική Μορφή

```
[IMAGE: url-εικόνας]
```

Πληκτρολογήστε αυτό σε **δική του γραμμή** (δική του παράγραφο) στον επεξεργαστή Blocks του Strapi.

### Πού βρίσκω το URL της εικόνας

1. Πηγαίνετε στη **Media Library** (Βιβλιοθήκη Πολυμέσων) στο Strapi
2. Κάντε κλικ στην εικόνα που θέλετε να χρησιμοποιήσετε
3. Αντιγράψτε το URL από τον πίνακα λεπτομερειών της εικόνας
   - Θα μοιάζει κάπως έτσι: `https://helpful-wealth-0a46a9eabb.media.strapiapp.com/filename.png`

### Απλό παράδειγμα

```
[IMAGE: https://helpful-wealth-0a46a9eabb.media.strapiapp.com/my-photo.png]
```

Αυτό εμφανίζει την εικόνα σε **μεσαίο μέγεθος**, **κεντραρισμένη** στη σελίδα.

## Πλήρης Μορφή (με επιλογές)

```
[IMAGE: url-εικόνας | εναλλακτικό κείμενο | μέγεθος | στοίχιση]
```

| Παράμετρος          | Υποχρεωτική; | Περιγραφή                              | Προεπιλογή |
|---------------------|--------------|----------------------------------------|------------|
| url-εικόνας         | Ναι          | Το πλήρες URL της εικόνας              | -          |
| εναλλακτικό κείμενο | Όχι          | Περιγραφή για προσβασιμότητα           | (κενό)     |
| μέγεθος             | Όχι          | Πόσο πλατιά εμφανίζεται η εικόνα      | medium     |
| στοίχιση            | Όχι          | Πού τοποθετείται η εικόνα στη σελίδα   | center     |

### Επιλογές μεγέθους

| Τιμή     | Πλάτος       | Κατάλληλο για                            |
|----------|--------------|------------------------------------------|
| `small`  | ~320px       | Λογότυπα, εικονίδια, μικρές εικόνες      |
| `medium` | ~512px       | Κανονικές φωτογραφίες, διαγράμματα       |
| `large`  | ~672px       | Πλατιές φωτογραφίες, banners             |
| `full`   | 100% πλάτος  | Banners πλήρους πλάτους, εικόνες ηρώων   |

### Επιλογές στοίχισης

| Τιμή     | Αποτέλεσμα                        |
|----------|-----------------------------------|
| `left`   | Εικόνα στοιχισμένη αριστερά       |
| `center` | Εικόνα κεντραρισμένη (προεπιλογή) |
| `right`  | Εικόνα στοιχισμένη δεξιά          |

### Εικόνα μόνο με εναλλακτικό κείμενο (μεσαίο μέγεθος, κεντραρισμένη)

## Σημαντικοί Κανόνες

1. **Δική της παράγραφος**: Η ετικέτα `[IMAGE: ...]` πρέπει να είναι **το μοναδικό κείμενο** στην παράγραφό της. Μην την αναμειγνύετε με άλλο κείμενο στην ίδια γραμμή.

2. **Χωρίς μορφοποίηση**: Μην εφαρμόζετε έντονη γραφή, πλάγια γραφή ή οποιαδήποτε άλλη μορφοποίηση στο κείμενο της ετικέτας. Πληκτρολογήστε το ως απλό κείμενο.

3. **Απαιτούνται αγκύλες**: Η ετικέτα ξεκινά με `[` και τελειώνει με `]`. Μην τις παραλείπετε.

4. **Διαχωριστικό pipe**: Χρησιμοποιήστε τον χαρακτήρα `|` (pipe / κάθετη γραμμή) για να διαχωρίσετε τις παραμέτρους. Μπορείτε να τον βρείτε στο πληκτρολόγιό σας:
   - **Mac**: `Option + 7` ή `Shift + \`
   - **Windows**: `AltGr + \` ή `Shift + \`

5. **Κενά γύρω από τα pipes**: Τα κενά πριν και μετά το `|` είναι προαιρετικά αλλά συνιστώνται για ευκολότερη ανάγνωση.

6. **Χωρίς διάκριση πεζών-κεφαλαίων**: Τα `[IMAGE:`, `[image:`, `[Image:]` λειτουργούν όλα με τον ίδιο τρόπο.

## Γρήγορη Αναφορά

```
Απλούστερη μορφή:
[IMAGE: url]

Με εναλλακτικό κείμενο:
[IMAGE: url | περιγραφή]

Με μέγεθος:
[IMAGE: url | περιγραφή | small]

Πλήρης έλεγχος:
[IMAGE: url | περιγραφή | large | right]
```

**Μεγέθη**: `small` | `medium` | `large` | `full`
**Στοίχιση**: `left` | `center` | `right`
