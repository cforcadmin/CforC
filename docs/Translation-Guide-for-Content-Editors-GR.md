# Οδηγός Μετάφρασης για Συντάκτες Περιεχομένου

## Επισκόπηση

Η ιστοσελίδα Culture for Change χρησιμοποιεί το **Google Translate** για αυτόματη μετάφραση περιεχομένου από Ελληνικά σε άλλες γλώσσες. Ωστόσο, η αυτόματη μετάφραση δεν παράγει πάντα ακριβή αποτελέσματα, ειδικά για:

- Ονόματα οργανισμών και ακρωνύμια
- Τεχνικούς όρους
- Κύρια ονόματα
- Εξειδικευμένο λεξιλόγιο

Αυτός ο οδηγός εξηγεί δύο εργαλεία που μπορείτε να χρησιμοποιήσετε για να ελέγξετε πώς μεταφράζεται το περιεχόμενό σας.

## Τα Δύο Εργαλεία Μετάφρασης

| Εργαλείο | Σκοπός | Ιδανικό Για |
|----------|--------|-------------|
| Σύνταξη `{λέξη}` | Αποτρέπει τη μετάφραση συγκεκριμένων λέξεων | Ακρωνύμια, εμπορικά σήματα, τεχνικούς όρους |
| EngTitle / EngDescription | Παρέχει πλήρη χειροκίνητη αγγλική μετάφραση | Ολόκληρους τίτλους ή περιγραφές που χρειάζονται ακριβή μετάφραση |

## Εργαλείο 1: Η Σύνταξη {λέξη}

### Τι Κάνει

Όταν περικλείετε μια λέξη ή φράση σε άγκιστρα `{έτσι}`, αυτό το κείμενο θα:
1. **ΔΕΝ μεταφράζεται** από το Google Translate
2. Εμφανίζεται ακριβώς όπως το πληκτρολογήσατε σε όλες τις γλώσσες
3. Τα ίδια τα άγκιστρα αφαιρούνται από την εμφάνιση

### Πώς να το Χρησιμοποιήσετε

Απλά περικλείστε οποιαδήποτε λέξη ή φράση θέλετε να προστατέψετε σε άγκιστρα:

```
Κανονικό κείμενο {προστατευμένο κείμενο} περισσότερο κανονικό κείμενο
```

### Παραδείγματα

#### Παράδειγμα 1: Προστασία Αγγλικού Ακρωνυμίου

**Τι πληκτρολογείτε στο Strapi:**
```
Το πρόγραμμα {S.I.M.A.} ξεκινά τον Φεβρουάριο
```

**Τι εμφανίζεται στην ιστοσελίδα (Ελληνικά):**
```
Το πρόγραμμα S.I.M.A. ξεκινά τον Φεβρουάριο
```

**Τι εμφανίζεται όταν μεταφραστεί στα Αγγλικά:**
```
The program S.I.M.A. starts in February
```

Παρατηρήστε: Το `S.I.M.A.` παραμένει ακριβώς το ίδιο, ενώ τα υπόλοιπα μεταφράζονται.

#### Παράδειγμα 2: Προστασία Εμπορικού Σήματος

**Τι πληκτρολογείτε στο Strapi:**
```
Συνεργασία με το {Culture for Change} και το {Creative Europe}
```

**Τι εμφανίζεται όταν μεταφραστεί:**
```
Cooperation with Culture for Change and Creative Europe
```

Τα ονόματα των οργανισμών παραμένουν αμετάβλητα.

#### Παράδειγμα 3: Προστασία Πολλαπλών Λέξεων

**Τι πληκτρολογείτε στο Strapi:**
```
Το {New European Bauhaus} είναι μια πρωτοβουλία της ΕΕ
```

**Τι εμφανίζεται όταν μεταφραστεί:**
```
The New European Bauhaus is an EU initiative
```

#### Παράδειγμα 4: Ελληνική και Αγγλική Έκδοση Μαζί

**Τι πληκτρολογείτε στο Strapi:**
```
Σύστημα Ηλεκτρονικής Μάθησης Ανοιχτής πρόσβασης – {Σ.Η.Μ.Α.}/{S.I.M.A.}
```

**Τι εμφανίζεται (Ελληνικά):**
```
Σύστημα Ηλεκτρονικής Μάθησης Ανοιχτής πρόσβασης – Σ.Η.Μ.Α./S.I.M.A.
```

**Τι εμφανίζεται (Αγγλικά):**
```
Open Access Electronic Learning System – Σ.Η.Μ.Α./S.I.M.A.
```

Και τα δύο ακρωνύμια προστατεύονται από τη μετάφραση.

## Εργαλείο 2: Πεδία Αγγλικής Παράκαμψης

### Τι Είναι

Στο Strapi, οι Δραστηριότητες και οι Ανοιχτές Προσκλήσεις έχουν προαιρετικά πεδία στα Αγγλικά:

- **EngTitle** - Αγγλική έκδοση του τίτλου
- **EngDescription** - Αγγλική έκδοση της περιγραφής

### Τι Κάνουν

Όταν ένας επισκέπτης βλέπει την ιστοσελίδα στα Αγγλικά (μέσω Google Translate):
- Αν το EngTitle είναι συμπληρωμένο → Εμφανίζει τον δικό σας αγγλικό τίτλο αντί της μετάφρασης του Google
- Αν το EngDescription είναι συμπληρωμένο → Εμφανίζει τη δική σας αγγλική περιγραφή αντί της μετάφρασης του Google

### Πώς να τα Χρησιμοποιήσετε

1. Πηγαίνετε στο **Content Manager** στο Strapi
2. Επιλέξτε μια **Δραστηριότητα** ή **Ανοιχτή Πρόσκληση**
3. Βρείτε το πεδίο **EngTitle** (κάτω από τον Τίτλο)
4. Βρείτε το πεδίο **EngDescription** (κάτω από την Περιγραφή)
5. Εισάγετε τη χειροκίνητη αγγλική μετάφρασή σας
6. **Αποθηκεύστε** και **Δημοσιεύστε**

### Σημαντικές Σημειώσεις

- Αυτά τα πεδία είναι **προαιρετικά** - αφήστε τα κενά αν η μετάφραση του Google είναι αποδεκτή
- Το EngDescription χρησιμοποιεί τον **επεξεργαστή εμπλουτισμένου κειμένου** (ίδιος με την Περιγραφή) - μπορείτε να προσθέσετε μορφοποίηση, συνδέσμους, κλπ.
- Μπορείτε να συνδυάσετε αυτό με τη σύνταξη `{λέξη}` και στα αγγλικά πεδία

## Πότε να Χρησιμοποιήσετε Ποιο Εργαλείο

### Χρησιμοποιήστε σύνταξη `{λέξη}` όταν:

- Έχετε μερικές συγκεκριμένες λέξεις που δεν πρέπει να μεταφραστούν
- Το υπόλοιπο κείμενο μπορεί να μεταφραστεί αυτόματα
- Προστατεύετε ακρωνύμια, ονόματα ή τεχνικούς όρους
- Γρήγορη και εύκολη λύση

### Χρησιμοποιήστε EngTitle/EngDescription όταν:

- Ολόκληρος ο τίτλος ή η περιγραφή χρειάζεται προσεκτική μετάφραση
- Το Google Translate παράγει κακά αποτελέσματα για ολόκληρο το κείμενο
- Το περιεχόμενο είναι σημαντικό και θέλετε πλήρη έλεγχο
- Έχετε χρόνο να γράψετε μια σωστή αγγλική έκδοση

### Χρησιμοποιήστε ΚΑΙ ΤΑ ΔΥΟ μαζί όταν:

- Παρέχετε αγγλική παράκαμψη αλλά περιέχει επίσης όρους που δεν πρέπει να μεταφραστούν σε άλλες γλώσσες
- Παράδειγμα: Το EngTitle σας περιέχει `{UNESCO}` για να το προστατέψετε όταν κάποιος μεταφράζει από Αγγλικά σε Γερμανικά

## Παραδείγματα Βήμα προς Βήμα

### Σενάριο 1: Νέα Δραστηριότητα με Προστατευμένο Ακρωνύμιο

**Εργασία:** Δημιουργία δραστηριότητας για το πρόγραμμα S.I.M.A.

1. Πηγαίνετε στο Content Manager → Activities → Δημιουργία νέας εγγραφής

2. **Πεδίο Title:**
   ```
   Νέο Πρόγραμμα {S.I.M.A.} για την Κοινωνία των Πολιτών
   ```

3. **Πεδίο EngTitle:** (προαιρετικό αλλά συνιστάται)
   ```
   New {S.I.M.A.} Program for Civil Society
   ```

4. **Πεδίο Description:**
   ```
   Το πρόγραμμα {S.I.M.A.} (System for Intelligent Management and Automation)
   στοχεύει στην ενίσχυση της κοινωνίας των πολιτών μέσω της τεχνολογίας.

   Οι συμμετέχοντες θα μάθουν για:
   - Ψηφιακά εργαλεία
   - Διαχείριση έργων
   - Συνεργατικές πλατφόρμες
   ```

5. **Πεδίο EngDescription:** (προαιρετικό)
   ```
   The {S.I.M.A.} program (System for Intelligent Management and Automation)
   aims to strengthen civil society through technology.

   Participants will learn about:
   - Digital tools
   - Project management
   - Collaborative platforms
   ```

6. Αποθηκεύστε και Δημοσιεύστε

### Σενάριο 2: Ανοιχτή Πρόσκληση με Ονόματα Οργανισμών

**Εργασία:** Δημιουργία ανοιχτής πρόσκλησης από το Creative Europe

1. Πηγαίνετε στο Content Manager → Open Calls → Δημιουργία νέας εγγραφής

2. **Πεδίο Title:**
   ```
   Πρόσκληση {Creative Europe} για Πολιτιστικά Έργα 2026
   ```

3. **Πεδίο EngTitle:**
   ```
   {Creative Europe} Call for Cultural Projects 2026
   ```

4. Συμπληρώστε τα άλλα απαιτούμενα πεδία (Προθεσμία, Περιγραφή, Σύνδεσμος, Εικόνα, κλπ.)

5. Αποθηκεύστε και Δημοσιεύστε

### Σενάριο 3: Δραστηριότητα με Σύνθετο Ελληνικό Ακρωνύμιο

**Εργασία:** Δραστηριότητα για το Σ.Η.Μ.Α. (ελληνικό ακρωνύμιο με αγγλικό ισοδύναμο S.I.M.A.)

1. **Πεδίο Title:**
   ```
   Εκπαιδευτικό Πρόγραμμα {Σ.Η.Μ.Α.} ({S.I.M.A.})
   ```

2. **Πεδίο EngTitle:**
   ```
   Educational Program {Σ.Η.Μ.Α.} ({S.I.M.A.})
   ```

Και το ελληνικό και το αγγλικό ακρωνύμιο προστατεύονται και στις δύο εκδόσεις.

## Βέλτιστες Πρακτικές

### ΚΑΝΕΤΕ:

1. **Πάντα προστατεύετε ονόματα οργανισμών**
   ```
   Συνεργασία με {UNESCO}, {UNICEF} και {Culture for Change}
   ```

2. **Προστατεύετε ονόματα προγραμμάτων/έργων**
   ```
   Το έργο {New European Bauhaus} χρηματοδοτείται από την ΕΕ
   ```

3. **Προστατεύετε τεχνικούς όρους που έχουν συγκεκριμένες σημασίες**
   ```
   Η πλατφόρμα χρησιμοποιεί {open source} λογισμικό
   ```

4. **Προστατεύετε διευθύνσεις email και URLs**
   ```
   Επικοινωνήστε στο {info@cultureforchange.gr}
   ```

5. **Παρέχετε EngTitle για σημαντικές ανακοινώσεις**
   - Προσκλήσεις χρηματοδότησης
   - Μεγάλες εκδηλώσεις
   - Δελτία τύπου

6. **Δοκιμάστε τις μεταφράσεις σας**
   - Μετά τη δημοσίευση, επισκεφθείτε τη σελίδα
   - Χρησιμοποιήστε το Google Translate για να αλλάξετε σε Αγγλικά
   - Επαληθεύστε ότι οι προστατευμένες λέξεις εμφανίζονται σωστά

### ΜΗΝ ΚΑΝΕΤΕ:

1. **Μην υπερχρησιμοποιείτε τη σύνταξη {λέξη}**
   - Προστατεύετε μόνο ό,τι χρειάζεται προστασία
   - Αφήστε το Google να μεταφράσει κοινές λέξεις

2. **Μην ξεχνάτε να κλείσετε τα άγκιστρα**
   ```
   Λάθος: {Culture for Change
   Σωστό: {Culture for Change}
   ```

3. **Μην φωλιάζετε άγκιστρα**
   ```
   Λάθος: {Το {UNESCO} πρόγραμμα}
   Σωστό: Το {UNESCO} πρόγραμμα
   ```

4. **Μην χρησιμοποιείτε άγκιστρα για μορφοποίηση**
   - Τα άγκιστρα είναι μόνο για έλεγχο μετάφρασης
   - Χρησιμοποιήστε τον επεξεργαστή εμπλουτισμένου κειμένου για έντονα, πλάγια, κλπ.

## Αντιμετώπιση Προβλημάτων

### Πρόβλημα: Τα άγκιστρα εμφανίζονται στην ιστοσελίδα

**Αιτία:** Το περιεχόμενο ίσως δεν περνάει από το component LocalizedText.

**Λύση:**
- Καθαρίστε την cache του browser σας
- Αν το πρόβλημα παραμένει, επικοινωνήστε με τον developer

### Πρόβλημα: Το αγγλικό πεδίο δεν εμφανίζεται όταν μεταφράζεται

**Αιτία:**
- Το πεδίο EngTitle/EngDescription μπορεί να είναι κενό
- Το περιεχόμενο μπορεί να μην είναι δημοσιευμένο

**Λύση:**
1. Ελέγξτε ότι το αγγλικό πεδίο έχει περιεχόμενο
2. Βεβαιωθείτε ότι κάνατε κλικ στο "Publish" (όχι μόνο "Save")
3. Καθαρίστε την cache του browser και ανανεώστε

### Πρόβλημα: Το Google Translate εξακολουθεί να μεταφράζει προστατευμένες λέξεις

**Αιτία:**
- Ξεχάσατε να προσθέσετε άγκιστρα
- Τυπογραφικό λάθος στα άγκιστρα

**Λύση:**
1. Επεξεργαστείτε το περιεχόμενο στο Strapi
2. Βεβαιωθείτε ότι η λέξη είναι περικλεισμένη: `{λέξη}` όχι `{λέξη` ή `λέξη}`
3. Αποθηκεύστε και Δημοσιεύστε

### Πρόβλημα: Μέρος του κειμένου λείπει

**Αιτία:** Μη κλεισμένα ή αναντίστοιχα άγκιστρα

**Λύση:**
1. Ελέγξτε ότι κάθε `{` έχει ένα αντίστοιχο `}`
2. Μην φωλιάζετε άγκιστρα μέσα σε άλλα άγκιστρα
