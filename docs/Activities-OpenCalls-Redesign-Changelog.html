Activities & Open Calls Redesign + Grid/List Toggle

Date: 2026-03-06 | Scope: Filter bar redesign, view toggles, category system, Strapi schema updates

Summary: Redesigned the Activities and Open Calls pages to match the Members page filter/search pattern. Added a reusable grid/list view toggle to all listing pages. Created shared filter components for consistent UI. Added a Funding Guidelines modal. Removed the Priority field from Open Calls and replaced it with a required Category field in Strapi. Assigned categories to all 63 activities and 73 open calls via keyword-matching scripts.
